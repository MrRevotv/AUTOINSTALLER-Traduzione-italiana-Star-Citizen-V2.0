## ✨ Caratteristiche del Launcher
Questo non è un semplice file, ma un gestore intelligente per la tua traduzione:

* 🎯 **Rilevamento Automatico:** Trova da solo la cartella di Star Citizen, anche se installata su dischi secondari.
* 📡 **Radar PTU:** La voce `PTU` appare nel menu solo quando la traduzione per quel server è effettivamente disponibile online.
* 🔄 **Aggiornamenti Real-Time:** Ti avvisa con una notifica di Windows non appena carichiamo una nuova versione della traduzione.
* 🎮 **Gioca Subito:** Una volta aggiornato, il pulsante diventa azzurro e ti permette di lanciare direttamente il gioco.
* ⚙️ **Personalizzazione:** Supporto per il caricamento di file `.ini` personalizzati tramite la funzione "Custom File".

## 🛠️ Installazione Automatica (Consigliata)

1. Clicca sul grande pulsante in alto per scaricare il file `StarCitizenItaLauncher.exe`.
2. Avvia il file scaricato.
3. **Avviso Windows SmartScreen:** Essendo un software indipendente, Windows potrebbe mostrare una schermata blu. Clicca su **`Ulteriori Informazioni`** e poi su **`Esegui Comunque`**. Il software è sicuro al 100%.

## 🎮 Come usare il Launcher

**1. Selezione Canale**
Il launcher cercherà il gioco sul tuo PC. Una volta trovato, seleziona dal menu a tendina il canale che vuoi tradurre (LIVE o PTU).

**2. Download e Installazione**
Se la traduzione manca o deve essere aggiornata, il pulsante diventerà verde. Cliccalo e il launcher scaricherà, rinominerà e configurerà tutto al posto tuo in pochi secondi.

**3. Gioca!**
A installazione completata, clicca su **"GIOCA"** per aprire il launcher ufficiale RSI e iniziare l'avventura.

---

## ⚙️ Impostazioni Avanzate
Tramite il tasto `⚙ SETTINGS` puoi:
* Impostare l'avvio automatico del launcher all'accensione del PC.
* Cambiare manualmente il percorso del gioco (se hai installazioni particolari).
* Abilitare il **Caricamento File Custom**: ti permetterà di usare un tuo file di traduzione tramite il pulsante viola `📂 CUSTOM`.

## 🖐️ Installazione Manuale (.zip)

Se preferisci non usare l'installer automatico, puoi scaricare il pacchetto ZIP aggiornato:

[![Scarica ZIP Manuale](https://img.shields.io/badge/DOWNLOAD-VERSIONE_MANUALE_(ZIP)-orange?style=for-the-badge&logo=archive)](https://github.com/MrRevotv/AUTOINSTALLER-Traduzione-italiana-Star-Citizen-V2.0/releases/latest/download/SC_Localizzazione_Italiano.zip)

> **Istruzioni:** Estrai il contenuto dello ZIP nella cartella principale del canale (es. `LIVE`). Assicurati che il file `user.cfg` sia nella root e che il file `global.ini` si trovi in `data\Localization\italian_(italy)\`.
